## Project Scope
This project is a simple weather application that shows current weather data for a searched city and provides a quick weather summary for common locations. The app uses Bootstrap for layout and styling, and fetches current weather information from a public weather API. The key scope items are:

- Search for a city and view its current temperature, humidity, wind speed, pressure, and condition.
- Display a summary card with temperature, weather condition, and humidity.
- Show weather data for common cities in a responsive table.
- Keep the interface clean and easy to use on both desktop and mobile.

## Files
- `index.html` - Main app markup and layout.
- `style.css` - Custom styles for the Weather App.
- `script.js` - JavaScript logic for fetching and displaying weather data.

## Notes
The app is designed to be easily extended with additional weather details or improved styling while preserving the current structure and functionality.
